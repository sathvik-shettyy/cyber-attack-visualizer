## Installation

### Clone the Repository

```bash
git clone https://github.com/sathvik-shettyy/cyber-attack-visualizer.git
cd cyber-attack-visualizer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Usage

### Running the Application

Ensure you have Python and pip installed. Follow the installation instructions mentioned above.

```bash
python app.py
```

### Accessing the Visualizer

Open your web browser and navigate to [http://127.0.0.1:5000/](http://127.0.0.1:5000/) to access the Cyber Attack Visualizer.
